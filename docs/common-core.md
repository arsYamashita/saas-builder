# Common Core

This project has fixed shared modules.

## Shared Modules
- Authentication
- Multi-tenant
- Role Based Access Control
- Stripe Billing
- Affiliate System
- Audit Logs
- Notification System
- Admin Layout

## Fixed Roles
- owner
- admin
- staff
- member
- affiliate_manager

## Important Rule
These modules are not regenerated for each project.
Only domain-specific entities and CRUD screens are generated.
