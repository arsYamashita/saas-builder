# simple_crm_saas Template Scope

## Purpose
This template is for:
- customer relationship management
- deal / opportunity tracking
- task management
- notes (optional in MVP)

## Included Domain Objects
- customers
- deals
- tasks
- notes (optional)

## Included Screens
- /dashboard
- /customers
- /customers/new
- /customers/[customerId]/edit
- /deals
- /deals/new
- /deals/[dealId]/edit
- /tasks
- /tasks/new
- /tasks/[taskId]/edit

## Included Core Modules
- auth
- tenant
- role based access control
- audit logs

## Explicitly Out of Scope
- affiliate tracking
- content publishing
- subscription billing (MVP: none)
- reservation / booking
- mobile app
- advanced analytics
- multi-language
- workflow builder
- email automation
- calendar sync
- payment processing
- notification system
