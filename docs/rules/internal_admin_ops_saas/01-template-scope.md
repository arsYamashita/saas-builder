# internal_admin_ops_saas Template Scope

## Purpose
This template is for:
- internal operation request management
- approval workflows
- category management
- audit trail

## Included Domain Objects
- operation_requests
- approvals
- categories

## Included Screens
- /dashboard
- /requests
- /requests/new
- /requests/[requestId]/edit
- /approvals
- /categories
- /categories/new

## Included Core Modules
- auth
- tenant
- role based access control
- audit logs

## Explicitly Out of Scope
- affiliate tracking
- content publishing
- subscription billing
- reservation / booking
- customer management
- deal management
- mobile app
- advanced analytics
- multi-language
- email automation
- file attachment
- SLA tracking
- calendar integration
