# membership_content_affiliate Template Scope

## Purpose
This template is for:
- membership site
- content publishing
- subscription billing
- affiliate tracking

## Included Domain Objects
- contents
- membership_plans
- subscriptions
- affiliates
- referrals
- commissions

## Included Screens
- /dashboard
- /content
- /content/new
- /content/[contentId]/edit
- /plans
- /plans/new
- /plans/[planId]/edit
- /billing
- /affiliate

## Included Core Modules
- auth
- tenant
- role based access control
- stripe billing
- affiliate tracking
- audit logs

## Explicitly Out of Scope
- mobile app
- advanced analytics
- multi-language
- workflow builder
- community feed
- DMs
- course progression
- quizzes
- file uploader with external media pipeline
