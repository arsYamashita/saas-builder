# Naming Rules

## Entity Names
Use these exact entity names:
- content
- membership_plan
- subscription
- affiliate
- referral
- commission

## Table Names
Use these exact table names:
- contents
- membership_plans
- subscriptions
- affiliates
- referrals
- commissions

## Route Segment Names
Use these exact route segments:
- content
- plans
- billing
- affiliate

## Variable Names
Use explicit names:
- contentId
- planId
- tenantId
- userId
- affiliateId
- referralId
- subscriptionId

Do not use:
- item
- data
- obj
- x
- y

## Component Names
Use these exact component names when relevant:
- ContentForm
- MembershipPlanForm
- AdminNav
- LogoutButton

## API Handler Function Names
Use standard exported handlers only:
- GET
- POST
- PATCH

Do not invent custom exported handler names.
